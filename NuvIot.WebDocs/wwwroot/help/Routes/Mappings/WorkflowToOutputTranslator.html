Workflow to Output Translator

The output from the Ouput Command is a normalized set of strongly typed name value pairs.
The job of the Output Translator is to take that normalized set of data convert it to Outgoing Messages that the Transmitter will send back to the device.

To enable this creation of outgoing messages you will need to map an Output Command to a message, then you will need to map the individual fields from the Output Command the Fields on the Message.
