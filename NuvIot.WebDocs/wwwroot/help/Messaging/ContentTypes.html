Content Types

NuvIoT is built to be flexible and work with your messages. To do so, we accept different types of content for your messages.

The following Message Content Types are supported:

  • Binary - A stream of bytes that represents the content of your message and can be parsed with our Binary Parser.
  • Delimited - A set of characters that are separated by some delimiter like a comma. Delimited Messages can be parsed with our Delimited Message Parser.
  • JSON - We support receving messages in JSON. JSON Messages can easily be parsed with our JSON Message Parser.
  • String - String messages can be parsed by either a regular expression or position with a string using our String Parser.
  • XML - XML Messages can be processed using XPath and our XML Message Parser.
