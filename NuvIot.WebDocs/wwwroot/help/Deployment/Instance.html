Instance

An Instance is a deployed version of a solution. Since NuvIoT was built for maximum reuse you configure a solution independently form how it is deployed. This allows for creating development and test environments as well as deploying the same solution in georedundent data centers. When you create an instance you select a configuration that will determine how your solution will scale.

Instances can also be either cloud based that runs on the NuvIoT Infrastructure or one that runs on the IoT Edge.
