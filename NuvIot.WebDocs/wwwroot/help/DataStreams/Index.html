Data Streams

Data Streams allow you to export data to be consumed by analysis engines or used to display charts and reports in the IoT Portal

Data Stream Types

The following connectors are provided for data streams.

  • AWS Elastic Search - Read and Write data to Amazons AWS Elastic Search
  • AWS S3 - Write data to Amazon Web Services S3 Storage
  • Azure Blob Storage - Write data to Azure Blob Storage
  • Azure Event Hubs - Write data to Azure Event Hubs
  • Azure Table Storage - Read and Write data to Azure Table Storage
  • SQL Server - Read and Write data to a Microsoft SQL Server
