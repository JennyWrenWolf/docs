Transmitter

Transmitters are used to take Outgoing Messages and send them to the originating device or a 3rd-party server or service. Applicable trasmitters include the following:

  • Azure Event Hub
  • Azure IoT Hub
  • Azure Service Bus
  • MQTT
  • Original Listener
  • REST Service
