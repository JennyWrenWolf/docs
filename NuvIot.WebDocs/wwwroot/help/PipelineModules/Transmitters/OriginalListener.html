Original Listener

If you need to send a message back to listener that originally received the message that initially generated this flow, you should choose the Origianl Listener Transmitter. This will take the message you generated and pass it back to the origianl listener.

Beta Alert! - Contact us if you need the following functionality: If the device is no longer connected, the message will be queued for delivery the next time the device connects.

Required Fields

In addition to the standard fields that are required for any transmitter, the following fields are required:

  • Transmitter Type (required)
    Origianl Listener
