Output Translator to Transmitter

The job of the Output Translator is to take the normalized output from the Ouput Command of the Workflow and turn it into the exact contents of the message that will be sent out with the Transmitter.

To map a message from the Outtranslator to the Transmitter, first use the Workflow to Output Mapping to map an Output Command to a message and then use this mapping to select which message(s) will be sent to the transmitter.
